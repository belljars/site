BASIC

BASIC (Beginner's All-purpose Symbolic Instruction Code) is a family of high-level programming languages designed for ease of use. It was originally developed in the mid-1960s and has since evolved into various dialects, influencing COMAL, Visual Basic, and GRASS.


Syntax Description
PRINT "Hello, World!" Outputs text to the screen.
LET A = 10 Assigns the value 10 to variable A.
IF A = 10 THEN PRINT "A is ten" Conditional statement.
FOR I = 1 TO 10: PRINT I: NEXT I Loop from 1 to 10, printing each value.
GOSUB 1000 Calls a subroutine located at line 1000.

BASIC syntax is, as the name implies, very basic, thus very easy to learn. Far more simpler than some more popular languages, such as Python or JavaScript. Thus reading and quickly understanding the code is straightforward, even for an amateur.


Code Examples


10 PRINT "Hello, World!"
20 GOTO 10

Prints Hello, World! indefinitely.
