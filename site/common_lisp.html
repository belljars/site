Common Lisp

Common Lisp is a powerful, multi-paradigm programming language that has been around since the 1980s. It is known for its flexibility, dynamic typing, and support for functional, procedural, and object-oriented programming styles. Common Lisp is often used in artificial intelligence research, symbolic computing, and rapid prototyping.

Syntax Description
(defun function-name (parameters) body) Defines a new function.
(let ((var1 value1) (var2 value2)) body) Creates local variables and evaluates the body.
(if condition then else) Conditional expression.
(cond (condition1 body1) (condition2 body2) ...) Multiple conditional branches.
(loop for var from start to end do body) Iterative loop construct.
(defclass class-name (superclasses) (slots)) Defines a new class.
(defmethod method-name ((arg1 class1) (arg2 class2)) body) Defines a method for a generic function.
(setf variable value) Sets the value of a variable.
(car list) Returns the first element of a list.
(cdr list) Returns the rest of the list after removing the first element.
(cons element list) Constructs a new list by adding an element to the front of an existing list.

Common Lisp uses S-Expression to represent code and data. The uniform structure makes it easy to manipulate code as data.


Code Examples


(defun square (x)
  (* x x))

(defun sum-of-squares (a b)
  (+ (square a) (square b)))

Defines a function square that returns the square of a number, and another function sum-of-squares that computes the sum of the squares of two numbers. 


(defclass person ()
  ((name :accessor name :initarg :name)
   (age :accessor age :initarg :age)))
(defmethod birthday ((p person))
  (setf (age p) (+ 1 (age p))))

Defines a class person with attributes name and age, and a method birthday that increments the age of the person.


Data types in Common Lisp

Data Type Description
Numbers Includes integers, rationals, floats, and complex numbers.
Strings Sequences of characters enclosed in double quotes.
Symbols Identifiers used to represent variables and functions.
Lists Ordered collections of elements, created using parentheses.
Arrays Multi-dimensional collections of elements.
Hash Tables Key-value pairs for efficient data retrieval.
Structures User-defined composite data types.
Classes and Objects Used in object-oriented programming to define blueprints and instances.
