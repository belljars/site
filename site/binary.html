Binary Code

Binary code is the fundamental language of computers, consisting of only two digits: 0 and 1. Each digit is called a bit, and a group of eight bits forms a byte. Binary code is used to represent all types of data in a computer, including numbers, text, images, and instructions for processing information.

Understanding Binary

In the binary system, each position represents a power of 2, starting from the rightmost position, which represents 2^0 (1). The next position to the left represents 2^1 (2), then 2^2 (4), and so on. To convert a binary number to decimal, you sum the values of the positions where there is a 1.

Braille

Braille is a tactile writing system used by people who are visually impaired. It consists of patterns of raised dots that represent letters, numbers, and punctuation marks. Each braille character is made up of up to six dots arranged in a 2x3 grid.

But how does braille relate to binary? Each dot in a braille cell can either be raised (1) or not raised (0), which can be represented using binary code. For example, the letter "A" in braille is represented by a single raised dot in the top-left position, which can be represented in binary as 100000.

ASCII


ASCII (American Standard Code for Information Interchange) is a character encoding standard that represents text in computers and other devices. It uses a 7-bit binary code to represent 128 different characters, including letters, numbers, punctuation marks, and control characters.
